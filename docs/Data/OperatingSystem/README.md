## 什么是操作系统？

简单来说 操作系统也是一个软件。它是管理下层硬件，为上层软件提供统一的、容易理解的抽象API的软件。



* 操作系统是一个控制程序
* 是一个资源管理器
  * 应用程序与硬件之间的中间层
  * 管理各种计算机软硬件资源
  * 提供访问计算机软硬件资源的高效手段
  * 解决资源访问冲突、确保资源公平使用
